My Project History

The experience of building this website can be described as ...

Honestly really rewarding. I've struggled and learned so much. I've never thought I would enjoy the front-end and designing a site. But creating this website has been so fun, and I've learned so much, especially from all of my mistakes and feedback from my teacher.

I'm so proud of the color scheme I've chosen for my site because I've never felt confident about my ability with visual design or any visual art. However, I thought I did well matching the vibe of Dungeons and Dragons. I'm also pleased that I get to work on something useful for my audience. There are barely any resources about signing various terms in D&D. I would want to keep updating and working on this website as I collect more resources for the Deaf community. Making this website responsive was probably the most challenging part of this project, so I'm proud of how hard I worked on that.
